# BDI KKU NMR AI Scientific Research & Benchmarks
=================================================

This folder contains scientific publications analysis, literature matrices, comparative tool diagnostics, and chemical shift benchmark resources used to ground our AI NMR spectrometer pipeline in physical chemistry.

---

## 📁 File Directory & Descriptions

| File Name | Language / Type | Description / Purpose |
|:---|:---|:---|
| [`literature_matrix.md`](literature_matrix.md) | English (Markdown) | Synthesizes scientific literature on deep learning methods for NMR peak alignment and denoising. |
| [`research01.md`](research01.md) | English (Markdown) | Deep dive into 1D Convolutional Neural Networks and transfer learning capabilities from electrocardiograms (ECG). |
| [`research02.md`](research02.md) | English (Markdown) | Explores continuous Neural Ordinary Differential Equations (Neural ODEs) for handling pH and temperature drifts in NMR. |
| [`research_gaps.md`](research_gaps.md) | English (Markdown) | Identifies limitations in current laboratory workflows and catalogs solutions enabled by our 4-stage pipeline. |
| [`tool_comparison.md`](tool_comparison.md) | English (Markdown) | Evaluates other academic NMR frameworks (such as NMRQNet and standard FastDTW libraries). |
| [`code_repos.md`](code_repos.md) | English (Markdown) | Curates open-source code repositories, model weights links, and chemical databases (HMDB/BMRB). |
