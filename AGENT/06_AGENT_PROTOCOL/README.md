# BDI KKU NMR AI Developer Protocols & Task Boards
===================================================

This folder contains repository coding conventions, agent development standards, technical decision logs, handoff guidelines, and active task tracking boards.

---

## 📁 File Directory & Descriptions

| File Name | Language / Type | Description / Purpose |
|:---|:---|:---|
| [`coding_standards.md`](coding_standards.md) | English (Markdown) | Architectural constraints, clean code rules, and type hinting guidelines. |
| [`writing_standards.md`](writing_standards.md) | English (Markdown) | Standards for documentation layouts, warning alerts, and visual diagram formatting. |
| [`agent_onboarding.md`](agent_onboarding.md) | English (Markdown) | Onboarding instructions detailing sandbox setup, environment checkouts, and runtime parameters. |
| [`decision_log.md`](decision_log.md) | English (Markdown) | Chronological log of critical design choices (e.g. choosing 512 latent dims, Pearson auto-classification, and EBM zone sizes). |
| [`task_board.md`](task_board.md) | English (Markdown) | Collaborative agile sprint board tracking active developer requirements. |
| [`handoff_template.md`](handoff_template.md) | English (Markdown) | Reusable markdown checklist for developer-to-developer or agent-to-agent session handoffs. |
