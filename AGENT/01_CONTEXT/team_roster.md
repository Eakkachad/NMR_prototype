## Role Definitions

### AI/ML Lead
- Designs and implements the ML pipeline (baseline + advanced)
- Writes Python code for preprocessing (binning, PQN), feature selection (PLS-DA VIP), and classification (SVM, Random Forest)
- Explores Neural ODE and EBM integration if time permits
- Validates model performance with metrics (accuracy, F1, confusion matrix)

### Data & Integration Lead
- Handles PDF data extraction and data engineering
- Builds the user-facing POC interface (web dashboard or notebook)
- Writes the proposal document
- Creates workflow diagrams and visual assets
- Designs for medical personnel usability

## Agent-to-Team Mapping

When AI agents work on this project, they act as **augmentation** for both team members:
- Coding tasks → primarily support AI/ML Lead
- Writing tasks → primarily support Data & Integration Lead
- Research tasks → support both
